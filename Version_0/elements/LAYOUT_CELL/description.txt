The LayoutCell component is a simple customizable layout,
it used Grid component from MUI to create a layout in the application.
and can be used xs (extra small), sm (small), md (medium), lg (large), xl (extra large) screen sizes.attr item, default is true and can not be change
Name Tag : KdpaGrid
children element: any element

static attrs:
1- xs: number
2- sm: number
3- md: number
4- lg: number
